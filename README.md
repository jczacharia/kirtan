# Orcha

Typescript-Native Client<->Serve Facilitator

## Features

- ✅ Type-safety between Client and Server
- ✅ Easy querying of highly-relational data
- ✅ Secure Endpoints and Queries
- ✅ Build-in pagination
- ✅ Singular and Multiple File Uploads
- ✅ Integration Testing
- ✅ Customizable User Authentication
- ✅ Realtime Subscriptions via WebSockets

### Current Compatible Technologies

- Nx
- Angular
- NestJS
- TypeORM
